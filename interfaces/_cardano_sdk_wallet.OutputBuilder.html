Interface OutputBuilder

partialOutput

partialOutput: Partial<Pick<TxOut, "address" | "datum"> & { value: Partial<Value> }>
  • Defined in packages/wallet/src/TxBuilder/types.ts:71

Transaction output that is updated by OutputBuilder methods. It should not be updated directly, but this is not restricted to allow experimental TxOutput changes that are not yet available in the OutputBuilder interface. Every method call recreates the partialOutput, thus updating it immutably.
